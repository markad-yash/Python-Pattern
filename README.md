# Python-Pattern
we will discuss about python code for pattern 


1. Triangle
```python
for i in range(6):
    for j in range(i):
        print("*",end=" ")
    print()
```
or
```python
for i in range(6):
    print(i*'* ')
```
Output:
```
* 
* *
* * *
* * * *
* * * * *
```
2. Square 
```python
for row in range(5):
    for column in range(5):
        print("*",end=" ")
    print()
```
Output:
```
* * * * * 
* * * * * 
* * * * *
* * * * *
* * * * *
```
